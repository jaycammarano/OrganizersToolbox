# Style Guide:

This project uses the AirBnB style guide. Linting and styling are enforced with ESLint and Prettier.

Addtional pages should be contained in the following file structure within the components folder:

```
|_*Page/
    |_ __test__
        |_className.spec.ts
        |_className2.spec.ts
    |_*Page.tsx
    |_interfaces.ts
    |_className.ts
```
